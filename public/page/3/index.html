LeetCode练习-Remove Nth Node From End of List

发表于 2019-06-24 分类于 算法日常练习

问题链接:Remove Nth Node From End of List

解法: 去除倒数第N个结点, 由于这是一个单链表, 无法从尾结点往前遍历. 那么可以将倒数第N个结点转为第L-N+1,那么思路就会清晰很多.

  1. 遍历两次, 第一次获取L,即链表的长度
  2. 第二次当遍历到L-N+1的位置时, 移除该结点即可
  3. 注意头结点和尾结点的特殊处理
