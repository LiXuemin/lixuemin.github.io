LeetCode 19. Remove Nth Node From End of List

发表于 2020-07-03 分类于 算法日常练习

题目

LeetCode 19. Remove Nth Node From End of List

思路

思路一:

  1. 单链表, 无法从尾结点往前遍历. 那么可以将倒数第N个结点转为第L-N+1个正数节点
  2. 长度L可以提前遍历一次来求值
  3. 移除第L - N + 1个节点

思路二:

  1. 利用双指针
  2. 先让两个指针相隔N - 1的位置
  3. 然后双指针同步后移
  4. 当第二个指针到了尾部, 第一个指针恰好在倒数第N个节点的前驱位置.
